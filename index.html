Браузер ранжирует селекторы по определенным правилам. Стиль применяется к тому селектору, весовой показатель которого выше.
  • * { } - универсальный селектор: 0 баллов
  • em, ::first-letter { } - селектор тега, псевдоэлемент: 1 балл
  • [title="Hello world!"], .one, :hover { } - селектор атрибута, класс, псевдокласс: 10 баллов
  • #header - идентификатор: 100 баллов
  • #header a:hover { } - идентификатор + тег + псевдокласс = 111 баллов
  • стили, прописанные в атрибуте или теге style: 1000 баллов
Правило применяется ко всем перечисленным селекторам.

Разница между идентификатором и классом заключается в том, что идентификатор может применяться для определения одного элемента, а класс может использоваться для определения нескольких элементов.
